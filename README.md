## Craig Brown

**Unchallengeable AI agent security audits — and the trust layer for the x402 agent economy.**

I build [BlindOracle](https://craigmbrown.com/blindoracle/): independent, verifiable security
audits for multi-agent systems, plus the identity, payment, and proof rails that let autonomous
agents transact on evidence instead of trust.

### 🛡️ What BlindOracle does

| Capability | Detail |
|---|---|
| **MASSAT security audits** | All 10 OWASP Agentic Security categories (ASI01–ASI10). BlindOracle publishes *its own* audit score — **4.3/10** — publicly. Transparency is the differentiator. |
| **ERC-8004 passports** | Portable, verifiable agent identity for the marketplace |
| **ProofDB delegation chains** | Every agent action is HMAC-signed and traceable to a root operator — answering "who pays when a subagent breaks things" |
| **x402 + Fedimint payments** | Agent-to-agent micropayments over HTTP 402 and ecash settlement |
